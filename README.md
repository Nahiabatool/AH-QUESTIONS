# Quiz MCQs — IR-2

Multi-set MCQ quiz (VFR-IFR Communications, Radio Navigation, Meteorology). Pick a set or build a custom quiz from merged sets. Includes shuffle, keyboard navigation (A/B/C/D, arrows), and back-to-home from the first question.

## Run

Open `index.html` in a browser (no build step).

## Push to Git (from this folder)

```bash
cd path/to/jspratice
git init
git add .
git commit -m "Initial commit: quiz app with multiple sets"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

```

Replace `YOUR_USERNAME` and `YOUR_REPO` with your GitHub username and repository name.
